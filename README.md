# Horizon Design System — Storybook

Build estático e publicado do Storybook do Design System Horizon
(componentes, tokens, identidade visual e pacote whitelabel para reuso em
outros projetos).

🔗 https://horizon-designsystem.vercel.app

Este repositório contém **apenas o build estático** (`storybook build`)
gerado a partir do projeto principal
[`horizon-suite`](https://github.com/guisantoshub/horizon-suite). Para
atualizar, gere um novo build (`npm run build-storybook`) no projeto
principal e republique o conteúdo aqui.
